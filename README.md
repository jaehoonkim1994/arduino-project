# Arduino LED Control using millis()

## Overview
Arduino Uno 환경에서 delay()를 사용하지 않고  
millis() 기반 타이머로 LED를 깜빡이게 만드는 프로젝트입니다.

Tinkercad Circuits 시뮬레이션 환경에서 구현했습니다.

## Key Concepts
- Non-blocking timing (millis)
- loop 기반 이벤트 처리
- 임베디드 시스템 기본 구조 이해

## Hardware
- Arduino Uno
- LED
- 220Ω Resistor
